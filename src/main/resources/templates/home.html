Trading Strategy Guide

Open Classification
IV (Inside Value) — 1st candle close between S1/PDL and R1/PDH
OV (Outside Value) — 1st candle close above R1/PDH or below S1/PDL
EV (Extended Value) — 1st candle close above R2 or below S2
Trend Following Setups
CPR level breakouts in the direction of the move:
BUY: CPR → R1/PDH → R2 → R3 → R4 (targets: next level up)
SELL: CPR → S1/PDL → S2 → S3 → S4 (targets: next level down)
IV/OV days: all breakouts active
EV days: only OR-aligned breakouts (continuation of gap)
Magnet Trades (IV/OV Days)
BUY above S1/PDL → target CPR (price pulled back to value)
SELL below R1/PDH → target CPR (price pulled back to value)
IV: full qty | OV: half qty (momentum may resist)
Mean Reversion (EV Days)
When OR is counter to gap direction, trade back toward value:
Gap Up + OR Bearish (sell reversals):
R4 → target R3 | R3 → target R2 | R2 → target R1/PDH | R1/PDH → magnet (CPR)
Gap Down + OR Bullish (buy reversals):
S4 → target S3 | S3 → target S2 | S2 → target S1/PDL | S1/PDL → magnet (CPR)
All reversals: half qty (counter daily trend)
Inside OR range on EV day: trade skipped
Quantity Stacking
1. Base qty (risk-based or fixed)
2. Capital cap
3. Extreme level (R2+ ÷2)
4. Session move % (÷2 if exceeded)
5. Probability (HPT 1x | MPT 0.5x | LPT 0.25x)
6. Level factor (R3/S3 0.75x | R4/S4 0.5x)
7. Mean-reversion (÷2 for EV reversals + OV magnets)
Min qty = 2 (even). Each step multiplies running qty.
